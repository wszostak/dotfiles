# dotfiles

My configurations.

## Install

**IMPORTANT:** This will *override* all existing files with a symlinks configured in `install` script.

```bash
$ sh -c "$(curl -fsSL https://raw.githubusercontent.com/wszostak/dotfiles/master/install)"
```
